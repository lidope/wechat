### 目录结构

~~~
└─images 服务器图片存放目录
└─view 小程序代码存放根目录
  ├─api 接口统一存放目录
  │ └─api.js 公共接口
  ├─components 组件目录
  │ └─iPhoneX 适配iPhoneX
  ├─css 全局css
  ├─images 图片目录
  ├─pages 小程序页面目录
  │ ├─demo 模板
  │ ├─home 首页
  │ │ └─index 首页
  ├─utils 工具类目录
  │ ├─autuLogin.js 自动授权登陆
  │ ├─request.js 请求基类
  │ ├─Server.js Socket类
  │ └─util.js 工具函数
  ├─wxParse WX分析类
  ├─app.js 入口文件
  ├─app.json 配置文件
  ├─app.wxss 样式
  ├─config.js 程序配置
  └─project.config.json 项目配置
~~~
